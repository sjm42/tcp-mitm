# tcp-mitm
Generic TCP man-in-the-middle with a side tap

```text
Usage: tcp_mitm [OPTIONS] --listen <LISTEN> --server <SERVER> --tap <TAP>

Options:
  -d, --debug            
  -t, --trace            
      --listen <LISTEN>  
      --server <SERVER>  
      --tap <TAP>        
  -h, --help             Print help
```
